# Security Policy

## Supported Versions

We currently support the following versions with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

We take the security of AI Swarm seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it privately.

Please send an email to [contact@ai-swarm.dev] with a description of the vulnerability, steps to reproduce, and potential impact.

### What to expect

- You will receive an acknowledgment of your report within 48 hours.
- We will provide an estimated timeframe for a fix.
- We will coordinate a public disclosure once the vulnerability is addressed.

Thank you for helping keep AI Swarm secure!
